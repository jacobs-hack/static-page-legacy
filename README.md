## Build (yarn) ##

### Requirements
* ruby
* compass
* node.js 
* git

### Running the site locally

Make sure to have the required apps running on you machine. Then execute
~~~bash
npm i -g yarn gulp serve
yarn install 
npm run build:dev
(on another terminal) npm run serve:dev
~~~


## Build (jekyll) ##

~~~bash
jekyll serve
~~~
